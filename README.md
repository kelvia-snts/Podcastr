### Technologies
Este projeto foi desenvolvido usando as seguintes tecnologias:

* Next.js
* React
* TypeScript

### Getting Started

CLone o projeto e acesse a pasta

$ git clone

E siga esses passos:

 * Instal the dependecies
$ yarn
 * Start the project
$ yarn dev 
* e
$ yarn server
